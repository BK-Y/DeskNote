# Plan-12-editor-05 — 右键菜单

## ① 核心问题
无法通过鼠标右键执行剪切/复制/粘贴。所有编辑操作只能通过键盘。

## ② 目标
- 右键点击弹出编辑菜单
- 有选区时：剪切、复制可用
- 无论有无选区：粘贴可用
- 菜单项和标准 Windows 右键行为一致

## ③ 方案设计
在 `WM_RBUTTONUP` 消息中根据鼠标位置决定菜单内容。使用 `TrackPopupMenu` 弹出标准 Windows 上下文菜单。剪切/复制/粘贴使用 `Clipboard` API 或内存缓冲区实现。

## ④ 实现细节
- 平台层转发 `WM_RBUTTONUP` 到 `App_OnRightButtonUp(x, y)`
- 根据 `Editor_GetSelection()` 决定菜单项启用/禁用
- 剪切：复制到剪贴板 + 删除选区
- 复制：`OpenClipboard → SetClipboardData(CF_UNICODETEXT) → CloseClipboard`
- 粘贴：`GetClipboardData → 写入`

命令 ID 映射：1004=剪切，1005=复制，1006=粘贴（与菜单已有的 1001/1002 规避冲突）。
