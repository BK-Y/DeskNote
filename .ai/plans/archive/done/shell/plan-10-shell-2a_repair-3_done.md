## 技术路线

在 `src/ui/titlebar.c` 中将 menu_button 的 label 改为：

```c
layout.menu_button.label = L"\u2630";  /* ☰ 汉堡图标 */
```

编译运行。然后逐项验证：

1. 右上角 ☰ 是否可见
2. 点击 ☰ 是否弹出菜单（含"关闭"和"关于"）
