# Plan-12-editor-06 — 多光标

## ① 核心问题
只能在一个位置编辑。无法同时在多个位置插入/删除文本。

## ② 目标
- 支持 Ctrl+点击添加额外光标
- 多光标下的插入、删除行为一致
- 多光标可视反馈（每个光标独立闪烁）

## ③ 方案设计
在 `Editor` 中维护光标列表（第一个光标为主光标），多光标编辑时对每个位置执行相同操作。Ctrl+点击在点击位置插入新光标，Esc 或点击空白清除多余光标。

## ④ 实现细节

### 光标模型
```c
typedef struct {
    int offset;       /* 文本偏移 */
    int visible;      /* 是否可见（用于闪烁） */
} EditorCursorItem;

typedef struct {
    EditorCursorItem cursors[8];  /* 最多 8 个光标 */
    int count;                     /* 当前光标数量 */
} EditorCursorSet;
```

### 编辑规则
- 主光标（索引 0）决定滚动位置和 IME
- 插入/删除时向每个光标位置写入/移除等量文本
- 撤销栈还原时恢复先前的多光标状态

### 重绘
`Render_DrawCarets` 遍历光标列表绘制每个光标。
