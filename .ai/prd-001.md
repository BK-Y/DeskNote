## 架构

### 文件结构

```
src/app/
  ├── app.h            ← AppState 增加 SysInfo sys + DnGlobal global
  ├── app.c            ← 编排：调 config.h API，直接访问 g_app.global.xxx
  ├── config.h         ← 新增，定义结构体 + 声明持久化 API
  └── config.c         ← 新增，实现持久化（内部调 sysinfo + state_store）

src/platform/win32/
  └── sysinfo.h/c      ← 新增，Win32 系统信息采集

src/config/
  ├── config.h         ← 旧 Config 层，标记废弃
  └── config.c         ← 旧 Config 层，标记废弃

src/storage/
  └── state_store.h/c  ← 不改动
```

### 结构体定义

```c
// SysInfo — 运行时采集，不持久化
typedef struct {
    int dpi;
    int scale_pct;               // dpi * 100 / 96
    RECT work_area;              // SPI_GETWORKAREA
    int screen_width;
    int screen_height;
} SysInfo;

// DnGlobal — 全局运行时配置，部分字段持久化
typedef struct {
    int resident_mode;           // 持久化
    int dock_edge;               // 持久化
    int dock_thickness;          // 运行时计算，不持久化
    int presence_state;          // 持久化
    int last_floating_left;      // 持久化
    int last_floating_top;       // 持久化
    int last_floating_width;     // 持久化
    int last_floating_height;    // 持久化
    int titlebar_height;         // 持久化
    int frame_visual_thickness;  // 持久化
    int release_on_drag_mode;    // 0=浮动置顶(默认) 1=普通窗口, 持久化
} DnGlobal;

// DnNote — 便签独立配置（本次仅定义框架）
typedef struct {
    wchar_t file_path[MAX_PATH];
} DnNote;
```

### 数据流

```
启动时:
  App_Init
    → SysInfo_Collect(hwnd, &g_app.sys)           // platform/sysinfo
    → g_app.global.dock_thickness = sys.screen_width * 15 / 100  // 运行时计算
    → StateStore_Load → 填充 g_app.global 其余字段   // app/config
    → g_app.shell 从 g_app.global 初始化             // app

运行时:
  直接读写 g_app.global.xxx
  模式切换后调 App_SaveGlobalState() → StateStore_Save

关闭时:
  App_Shutdown → App_SaveGlobalState()
```

### dock_thickness 生命周期

```
启动 → 由 sys.screen_width * 15 / 100 计算
        ↓
菜单切换贴边 → 用当前 g_app.global.dock_thickness（内存值，保持启动时的计算值）
        ↓
分辨率变化(WM_DISPLAYCHANGE) → 重新采集 SysInfo → 重算 dock_thickness
        ↓
不写入 state.ini，不持久化
```

### 持久化对照

| 字段 | 持久化？ | 说明 |
|:----:|:--------:|------|
| resident_mode | ✅ | 用户偏好，写 state.ini |
| dock_edge | ✅ | 用户偏好，写 state.ini |
| dock_thickness | ❌ | 每次启动从 sys.screen_width 重算 |
| presence_state | ✅ | 写 state.ini |
| last_floating_* | ✅ | 写 state.ini |
| titlebar_height | ✅ | 写 state.ini |
| frame_visual_thickness | ✅ | 写 state.ini |
| release_on_drag_mode | ✅ | 用户配置，写 state.ini |
