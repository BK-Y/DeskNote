# DeskNote 开发计划 - 总览

## 项目定位

轻量、高效的桌面便签软件，支持 Markdown 部分标签、TODO 管理、跨平台。

```
最终产物：单个 desknote.exe (~300KB)，绿色免安装
技术栈：  C语言 + Win32 API + md4c + Rich Edit
跨平台：  后续阶段 10 用宏封装平台差异
```

## 版本号规则

遵循标准语义化版本 `major.minor.patch`：

| 版本 | 含义 | 当前 |
|------|------|------|
| `0.x.x` | 开发阶段，API 不稳定 | ✅ `0.1.0` |
| `1.x.x` | 首个稳定版发布 | |
| `x.0.0` | major：大版本重构，不兼容旧版 | |
| `x.1.x` | minor：新增功能，向下兼容 | |
| `x.x.1` | patch：Bug 修复，功能不变 | |

| 阶段 | 建议版本 | 说明 |
|------|----------|------|
| 0～2 | `0.1.x` | 基础窗口 + 编辑功能 |
| 3～5 | `0.2.x` | 持久化 + 托盘 |
| 6～8 | `0.3.x` | 多便签 + TODO |
| 9～10 | `0.4.x` | 美化 + 跨平台 |
| 11 | `0.5.x` | Vim 模式 + 设置 |

## 自定义标签语法

### 时间标签
| 标签 | 含义 | 示例 |
|------|------|------|
| `@YYYY-MM-DD` | 截止日期 | `@2025-01-20` |
| `@YYYY-MM-DD HH:MM` | 截止日期+时间 | `@2025-01-18 14:00` |
| `@today` | 今天 | 每天自动对齐 |
| `@tomorrow` | 明天 | |
| `@next-mon` `@next-fri` | 下周一 / 下周五 | |
| `@every day` `@every fri` | 周期重复 | 完成后自动追加新任务 |

### 优先级标签
| 标签 | 含义 |
|------|------|
| `!high` | 高优先级（红色） |
| `!medium` | 中优先级（橙色） |
| `!low` | 低优先级（灰色） |

### 分类标签
| 标签 | 含义 |
|------|------|
| `#tag` | 行内分类（注意：`#` 后无空格才是标签，有空格是标题） |

### 勾选框
| 标签 | 含义 |
|------|------|
| `- [ ]` | 未完成任务 |
| `- [x]` | 已完成任务 |

### 便签示例

```markdown
# 购物清单

- [x] 牛奶 #日用品 @2025-01-15
- [ ] 鸡蛋 #食品 @tomorrow
- [ ] 洗衣液 #日用品 @2025-01-20 !low
- [ ] 买生日礼物 @2025-01-19 !high

## 工作

- [ ] 周报 #工作 @every fri 17:00 !high
- [ ] 整理桌面文件 @today !low
```
